# 校园二手交易系统 (Campus Second-Hand Trading System)

软件工程课程实训项目 | 前后端分离架构

## 项目简介

本项目是面向本校在校大学生设计开发的校内闲置物品交易平台。

旨在解决学生教材、数码产品、生活用品、体育器材等闲置资源浪费问题，提供安全、高效、垂直化的校园内部交易闭环。

项目完整遵循软件工程开发流程（需求分析 -> 设计 -> 编码 -> 测试 -> 维护），是《软件工程》课程的综合性实训成果，覆盖前后端分离开发、数据库设计与团队协作等核心能力。

## 核心功能模块

### 1. 用户中心 (User)

- 用户注册、账号登录（JWT认证）
- 个人信息编辑、头像上传
- 学生身份认证（可选）
- 我的发布、我的收藏、我的订单统一管理

### 2. 商品交易 (Commodity)

- 商品发布：支持多图上传、标题、详情、价格、成色选择
- 商品管理：编辑、下架、删除（逻辑删除）
- 智能检索：分类筛选、关键词搜索、价格区间排序
- 商品咨询：买卖双方在线沟通或留言互动

### 3. 订单系统 (Order)

- 订单创建与状态流转（待售/已售/已取消）
- 校内自提记录管理
- 订单评价与反馈

### 4. 后台管理 (Admin)

- 用户账号封禁与管理
- 商品审核与违规下架
- 数据统计与图表展示

## 技术栈选型

### 前端 (Frontend)

| 类别 | 选型 | 备注 |
| :--- | :--- | :--- |
| 核心框架 | Vue 3 | 组合式 API (Composition API) |
| UI 组件 | Element Plus | 基于 Vue 3 的桌面端组件库 |
| 请求工具 | Axios | 异步 HTTP 客户端 |
| 构建工具 | Vite | 极速热更新 |

### 后端 (Backend)

| 类别 | 选型 | 备注 |
| :--- | :--- | :--- |
| 核心框架 | Spring Boot 2.x | 快速开发脚手架 |
| 持久层 | MyBatis-Plus | 简化 CRUD 操作 |
| 安全框架 | Spring Security | 处理认证与授权 |
| 接口规范 | RESTful API | 标准化接口设计 |

### 数据与工具

- **数据库**：MySQL 8.0
- **缓存**：Redis（可选）
- **构建工具**：Maven 3.6+
- **开发环境**：IntelliJ IDEA + VS Code
